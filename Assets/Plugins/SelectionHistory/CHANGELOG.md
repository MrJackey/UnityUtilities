# 1.1.0 (2026-06-24)

### Features
- Added the ability to drag 'n drop items directly from the history window
- Changed the default response when navigating history from pinging to selecting the object
- Added a button in the history window to ping objects in the editor
- Changed the history to only include the latest of any duplicates
- Added separate setting to filter out folders when including assets
- Increased history size (10 to 25)
